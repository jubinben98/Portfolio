Handwritten Number Generator


  • Tech Stack: Python, OpenCV and Numpy
  • Github URL: Project Link

The MNIST Digits Sequence Generator is a Python package that enables the generation of sequences of digits and the creation of datasets containing images resembling Japanese phone numbers. This package provides both a Python API and command-line tools for easy usage.

Installation: You can install the package with pip install number-generator.

Python Package: The core function generate_numbers_sequence generates images based on provided digits, spacing, and width parameters.

Command-Line Interface (CLI): Two CLI scripts are available. generate-numbers-sequence generates digit sequences as images, while generate-phone-numbers creates datasets of phone number-like images.

Documentation: Detailed instructions on installation, usage, and implementation details can be found in the Project ReadMe.

Quality and Testing: The project follows best practices in code organization, extensibility, and includes thorough documentation and tests for correctness.

Extras: Additional features include adding noise to images and different writing styles for phone numbers. Docker support with multiscale builds is also available.

Feel free to explore and contribute to this project to enhance its functionality.
